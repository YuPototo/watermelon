# 2 Prettier and ESlint

我对 Prettier 和 ESlint 的使用按照如下规则：

-   保存代码时，使用 Prettier 自动 format
-   Disable ESlint 插件
-   在需要 lint 时，再运行 `npx eslint dir_or_file`
